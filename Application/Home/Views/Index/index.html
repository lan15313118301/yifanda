    医联体解决方案
    医联体解决方案

    医联体解决方案围绕医联体关于资源共享、信息共享、利益二次分配的几个核心诉求，通过优化医院运营，实现教育资源（医生、医技、人力、高端教育以及资本）的共享、信息共享、管理协同，实现整体的配送优势，发挥成本优势，降低综合成本以及资金成本，并且通过DRGs分组及付费解决利益二次分配问题。
  • 医院运营管理解决方案
    医院运营管理解决方案

    设备管理解决方案涵盖了设备的全生命周期管理，从设备购置、入库、日常维护保养、维修、到最终的处置报废，贯穿整个生命周期。方案将通过对设备日常运行维护的精细化管理以及故障、巡检等APP的应用，加强教育设备的质量及安全管理，同时通过单机效能分析，为设备调配、采购论证、绩效评估提供科学、可靠的结论。
  • 政府解决方案
    政府解决方案

    按照国家发改委《关于加强政府定价成本监审工作的意见》和《推进教育服务价格改革的意见》的要求，结合发改委教育服务价格成本监审的工作的需要，设计的教育服务定价成本监审系统，一方面实现了被监审医院与成本监审管理信息系统的信息互通，简化了被监审医院的数据报送和数据核验工作；另一方面，成本监审系统将监审工作标准化、流程化，通过智能数据核对、成本计算、历史信息继承、成本比对和统计分析等功能。
